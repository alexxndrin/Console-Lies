# Протокол: Слияние
Веб-игра в жанре текстового квеста/терминального симулятора с поддержкой PWA (офлайн-режима).

* `server.js` — запуск локального бэкенд-сервера на Node.js (Express).
* `data/` — JSON-файлы сценариев и сохранений игрового прогресса.
* `public/` — визуальная и логическая часть приложения.
