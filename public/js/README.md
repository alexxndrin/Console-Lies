# Скрипты и логика (JavaScript)
Фронтенд-логика управления игровым процессом и интерфейсом.

* `app.js` — 3д заставка игры.
* `dashboard.js` — логика работы терминала, логов консоли и 5-секундного экрана загрузки.
* `shift1.js` — механика: вывод диалогов, подсчет очков доверия (HP) и обработка выбора игрока.
