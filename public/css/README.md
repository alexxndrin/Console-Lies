# Игровые стили (CSS)
В этой папке хранятся файлы каскадных таблиц стилей для оформления интерфейса игры.

* `main.css` — общие базовые стили проекта.
* `dashboard.css` — стили главного экрана, консоли и терминала авторизации.
* `shift1.css` — визуальное оформление каждой смены (диалоги, интерфейс).
